## 1. 概述

### 1.1 功能定位

"我的综价"是个人数据仓库中的清单综合单价管理中心，负责：
- **汇聚**两个来源的综价数据（造价文件解析 + 综价文件上传）
- 提供综价数据的查看、筛选、分析
- 支持综价数据的PR补录（补充信息、调整标准化映射）
- **推送到企业库**（综价库）

### 1.2 数据来源与汇聚

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   数据来源1：造价文件                      我的综价                          │
│   ┌─────────────────┐                                                      │
│   │                 │                     ┌───────────────────────────────┐ │
│   │  造价文件采集    │                     │                               │ │
│   │       ↓         │                     │  清单综价数据汇聚              │ │
│   │  分析（标准化）  │ ─── 推送 ──────────▶│                               │ │
│   │       ↓         │   综价数据           │  ┌───────────────────────┐   │ │
│   │  我的项目       │                     │  │ 来源      │ 数量       │   │ │
│   │                 │                     │  │───────────│───────────│   │ │
│   └─────────────────┘                     │  │ 造价文件   │ 1,340 条  │   │ │
│                                           │  │ 综价文件   │ 528 条    │   │ │
│   数据来源2：综价文件                      │  │───────────│───────────│   │ │
│   ┌─────────────────┐                     │  │ 合计      │ 1,868 条  │   │ │
│   │                 │                     │  └───────────────────────┘   │ │
│   │  综价文件采集    │ ─── 推送 ──────────▶│                               │ │
│   │  （独立上传）    │   综价数据           │  统一标准化：SBoQCode         │ │

### 1.3 状态流转

```
┌──────────┐                  ┌──────────┐                  ┌──────────┐
│  待补录   │ ── PR补录 ────▶ │  已补录   │ ── 推送PR ────▶ │  已推送   │

| 状态 | 说明 | 可执行操作 |

## 9. 与我的材价的对比

### 9.1 结构对比

| 维度 | 我的材价 | 我的综价 |
|------|----------|----------|
| 数据内容 | 材料单价 | 清单综合单价 |
| 标准化目标 | SMLCode + ParamMap | SBoQCode + FeatureMap |
| 单价组成 | 无 | 人工费+材料费+机械费+管理费+利润 |
| 额外字段 | 品牌、税率、运费 | 计价阶段、专业分类、项目特征 |
| 推送目标 | 材价库 | 综价库 |

### 9.2 共同点

| 维度 | 说明 |
|------|------|
| 数据来源 | 造价文件解析 + 独立文件上传 → 汇聚 |
| 状态流转 | 待补录 → 已补录 → 已推送 |
| 操作按钮 | 查看、PR补录、推送、删除 |
| 批量功能 | 批量推送、批量删除、导出Excel |
| PR审核 | 推送到企业库需要PR审核 |
