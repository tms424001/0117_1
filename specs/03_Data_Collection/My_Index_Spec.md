## 1. 概述

### 1.1 功能定位

"我的指标"是个人数据仓库中的指标管理中心，负责：
- 自动接收"我的项目"生成的指标数据
- 提供指标数据的查看、筛选、分析
- 支持指标数据的PR补录（补充说明、调整标签）
- **推送到企业库**（指标库）

### 1.2 数据来源

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   我的项目                                        我的指标                   │
│   ┌─────────────────┐                           ┌─────────────────────────┐ │
│   │                 │                           │                         │ │
│   │  标签化处理      │                           │  项目级指标              │ │
│   │       ↓         │     自动同步               │  ├─ 经济指标            │ │
│   │  计算指标 ───────┼──────────────────────────▶│  ├─ 工程量指标          │ │
│   │                 │                           │  └─ 人材机指标          │ │
│   │                 │                           │                         │ │
│   └─────────────────┘                           │  单位工程指标            │ │
│                                                 │  ├─ 门诊楼-土建          │ │
│                                                 │  ├─ 门诊楼-装饰          │ │
│                                                 │  └─ ...                 │ │
│                                                 │                         │ │
│                                                 └─────────────────────────┘ │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 1.3 指标层级

| 层级 | 说明 | 数据来源 |
|------|------|----------|
| 项目级指标 | 整个项目的汇总指标 | 我的项目计算生成 |
| 单位工程指标 | 每个单位工程的指标 | 我的项目计算生成 |

### 1.4 状态流转

```
┌──────────┐                  ┌──────────┐                  ┌──────────┐
│  待补录   │ ── PR补录 ────▶ │  已补录   │ ── 推送PR ────▶ │  已推送   │
└──────────┘                  └──────────┘                  └──────────┘
      │                             │
      └─────── 直接推送 ────────────┘
```

| 状态 | 说明 | 可执行操作 |
|------|------|------------|
| pending | 待补录 | 查看、PR补录、推送、删除 |
| completed | 已补录 | 查看、PR补录、推送、删除 |
| pushed | 已推送 | 查看、删除 |

### 2.2 列表字段定义

| 字段 | 说明 | 宽度 | 排序 |
|------|------|------|------|
| 复选框 | 批量操作选择 | 40px | - |
| 展开/折叠 | 展开显示单位工程指标 | 30px | - |
| 项目/单位工程 | 项目名称或单位工程名称 | 自适应 | 支持 |
| 功能标签 | 匹配的功能标签 | 100px | 支持 |
| 规模档 | 规模分档（仅项目级显示） | 60px | 支持 |
| 地点 | 项目所在地（仅项目级显示） | 60px | 支持 |
| 单方造价 | 元/m² | 80px | 支持 |
| 钢筋含量 | kg/m²（土建才有） | 80px | 支持 |
| 状态 | 当前状态 | 80px | 支持 |
| 操作 | 操作按钮组 | 160px | - |

### 2.3 操作按钮

| 按钮 | 图标 | 说明 | 适用范围 |
|------|------|------|----------|
| 查看 | 👁 | 查看指标详情 | 全部 |
| PR补录 | ✏️ | 补录指标说明 | 待补录、已补录 |
| 推送 | 📤 | 推送到指标库 | 待补录、已补录 |
| 删除 | 🗑 | 删除指标 | 待补录、已补录 |

**按钮状态规则：**

| 状态 | 查看 | PR补录 | 推送 | 删除 |
|------|:----:|:------:|:----:|:----:|
| 待补录 | ✓ | ✓ | ✓ | ✓ |
| 已补录 | ✓ | ✓ | ✓ | ✓ |
| 已推送 | ✓ | ✗ | ✗ | ✓ |

## 9. 与我的项目的关系

### 9.1 数据同步关系

```
我的项目                                        我的指标
┌─────────────────────┐                        ┌─────────────────────┐
│                     │                        │                     │
│  项目：XX医院门诊楼  │                        │  项目级指标          │
│                     │     计算指标完成        │  ├─ 经济指标        │
│  单位工程：         │ ─────────────────────▶ │  ├─ 工程量指标      │
│  ├─ 门诊楼-土建     │     自动同步            │  └─ 人材机指标      │
│  ├─ 门诊楼-装饰     │                        │                     │
│  └─ 门诊楼-安装     │                        │  单位工程指标        │
│                     │                        │  ├─ 门诊楼-土建     │
│                     │                        │  ├─ 门诊楼-装饰     │
│                     │                        │  └─ 门诊楼-安装     │
│                     │                        │                     │
└─────────────────────┘                        └─────────────────────┘
```

### 9.2 状态关联

| 我的项目状态 | 我的指标状态 | 说明 |
|-------------|-------------|------|
| 待标签化 | - | 尚未生成指标 |
| 待计算指标 | - | 尚未生成指标 |
| 已生成指标 | 待补录 | 指标自动同步 |
| 已补录 | 待补录/已补录 | 项目补录不影响指标状态 |
| 已推送 | 已推送 | 项目推送时可同步推送指标 |

### 9.3 独立推送

我的指标支持**独立于我的项目进行推送**：
- 我的项目可以只推送到案例库
- 我的指标可以单独推送到指标库
- 也可以在我的项目推送时同步推送指标
