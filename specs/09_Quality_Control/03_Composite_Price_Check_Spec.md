## 1. 功能概述

### 1.1 功能定位

综合单价检查是质控模块的**核心检查能力**，通过多维度校验确保清单项综合单价的准确性和合理性。综合单价是清单计价的核心，直接决定工程造价的准确性。

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          综合单价检查定位                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   综合单价构成                                                          │
│   ────────────                                                          │
│   ┌─────────────────────────────────────────────────────────┐          │
│   │                     综合单价                             │          │
│   │  ┌─────────┬─────────┬─────────┬─────────┬─────────┐   │          │
│   │  │ 人工费  │ 材料费  │ 机械费  │ 管理费  │  利润   │   │          │
│   │  │ 15-25% │ 55-70% │  5-10% │  8-15% │  3-8%  │   │          │
│   │  └─────────┴─────────┴─────────┴─────────┴─────────┘   │          │
│   └─────────────────────────────────────────────────────────┘          │
│                                                                         │
│   三维检查模型                                                          │
│   ──────────────                                                        │
│   ┌─────────────┐  ┌─────────────┐  ┌─────────────┐                   │
│   │   合理性    │  │   一致性    │  │   完整性    │                   │
│   │  Validity   │  │ Consistency │  │ Completeness│                   │
│   ├─────────────┤  ├─────────────┤  ├─────────────┤                   │
│   │ 单价是否在  │  │ 相同清单项  │  │ 人材机组成  │                   │
│   │ 历史区间内  │  │ 单价是否    │  │ 是否完整   │                   │
│   │ （对比综价库）│  │ 一致       │  │ 合理       │                   │
│   └─────────────┘  └─────────────┘  └─────────────┘                   │
│                                                                         │
│   与材料价格检查的区别                                                  │
│   ────────────────────                                                  │
│   • 材料价格检查：检查单一材料的价格                                   │
│   • 综合单价检查：检查清单项的组合价格（包含人材机+管理费+利润）       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.2 检查前提

**标准化是检查的基础**：综合单价检查依赖标准化引擎，只有将清单项归一化到 SBoQCode 后，才能进行有效的单价比对。

```
原始清单名称                标准化处理                 可比对
────────────                ──────────                 ──────
"C30矩形柱"        ─┐                          ┌─▶  综价库历史数据
"矩形柱(C30砼)"    ─┼─▶  SBoQCode: 010501004  ─┼─▶  同类项目单价
"现浇C30混凝土柱"  ─┘     + FeatureMap         └─▶  文件内其他同类项
```

### 1.3 检查维度总览

| 检查维度 | 检查目的 | 数据依赖 | 难度 |
|----------|----------|----------|:----:|
| **合理性检查** | 单价是否在合理区间 | 综价库历史数据 | ★★★ |
| **一致性检查** | 文件内相同清单项单价是否一致 | 文件内数据 | ★★☆ |
| **完整性检查** | 单价组成是否完整、占比是否合理 | 行业经验规则 | ★★☆ |

## 3. 合理性检查

### 3.1 检查原理

合理性检查通过将文件中的综合单价与**历史数据**进行对比，判断单价是否在合理浮动范围内。

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          合理性检查原理                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   基准单价来源（优先级从高到低）                                        │
│   ──────────────────────────────                                        │
│                                                                         │
│   ┌─────────────────┐                                                   │
│   │ 1. 企业综价库    │  ← 企业历史成交数据，最可靠                      │
│   ├─────────────────┤                                                   │
│   │ 2. 平台综价库    │  ← 平台汇集的行业数据                            │
│   ├─────────────────┤                                                   │
│   │ 3. 定额测算价    │  ← 按定额标准测算的理论价格                      │
│   ├─────────────────┤                                                   │
│   │ 4. 指标反算价    │  ← 按指标库反算的参考价格                        │
│   └─────────────────┘                                                   │
│                                                                         │
│   匹配条件                                                              │
│   ────────                                                              │
│   • SBoQCode（清单编码）必须匹配                                       │
│   • 项目特征应相近（如混凝土强度、截面尺寸）                           │
│   • 计价阶段应匹配（招标/投标/结算）                                   │
│   • 地区应相近（同省/同市优先）                                        │
│   • 时间应相近（近1-2年数据）                                          │
│                                                                         │
│   浮动区间计算                                                          │
│   ────────────                                                          │
│                                                                         │
│   参考价格通常取：中位数 或 加权平均值                                  │
│   合理区间 = [参考价 × (1 - 下浮比例), 参考价 × (1 + 上浮比例)]        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3.2 基准单价匹配

#### 3.2.1 匹配流程

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        基准单价匹配流程                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  ┌─────────────────┐                                                    │
│  │ 文件中的清单项  │                                                    │
│  │ 编码+名称+特征  │                                                    │
│  └────────┬────────┘                                                    │
│           │                                                             │
│           ▼                                                             │
│  ┌─────────────────┐                                                    │
│  │  标准化引擎     │  清单 → SBoQCode                                  │
│  │  归一化处理     │  特征 → FeatureMap                                │
│  └────────┬────────┘                                                    │
│           │                                                             │
│           ▼                                                             │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                    综价库查询                                    │   │
│  │                                                                  │   │
│  │  查询条件：                                                      │   │
│  │  • SBoQCode（清单编码）                                         │   │
│  │  • 项目特征（强度等级、规格尺寸等）                             │   │
│  │  • 计价阶段（招标/投标/结算）                                   │   │
│  │  • 地区（省/市）                                                │   │
│  │  • 时间范围（近2年）                                            │   │
│  │                                                                  │   │
│  │  查询策略：                                                      │   │
│  │  ① 精确匹配（SBoQCode + 特征 + 阶段 + 地区）                   │   │
│  │  ② 特征放宽（SBoQCode + 阶段 + 地区，特征相近）                │   │
│  │  ③ 地区放宽（SBoQCode + 特征 + 阶段，上级地区）                │   │
│  │  ④ 时间放宽（SBoQCode + 特征 + 地区，扩大时间范围）            │   │
│  │                                                                  │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│           │                                                             │
│           ▼                                                             │
│  ┌─────────────────┐                                                    │
│  │  计算参考价格   │  中位数 / 加权平均 / 分位数                       │
│  │  确定合理区间   │  参考价 × (1 ± 浮动比例)                         │
│  └─────────────────┘                                                    │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

#### 3.2.2 特征相似度计算

不同清单项的特征对单价影响程度不同，需要计算特征相似度：

| 清单类型 | 关键特征 | 权重 | 相似度计算 |
|----------|----------|:----:|-----------|
| 混凝土工程 | 强度等级 | 40% | 等级差≤1级为高相似 |
| 混凝土工程 | 截面尺寸 | 30% | 尺寸差≤20%为高相似 |
| 混凝土工程 | 构件类型 | 30% | 同类型为完全匹配 |
| 钢筋工程 | 钢筋种类 | 40% | 必须完全匹配 |
| 钢筋工程 | 直径规格 | 40% | 规格差≤2级为高相似 |
| 砌筑工程 | 砌体材料 | 50% | 必须完全匹配 |
| 砌筑工程 | 墙厚 | 30% | 厚度差≤50mm为高相似 |
| 砌筑工程 | 砂浆种类 | 20% | 等级差≤1级为高相似 |

#### 3.3.1 按清单类别配置

| 清单类别 | 默认上浮 | 默认下浮 | 说明 |
|----------|:--------:|:--------:|------|

| 招标控制价 | ±15%~20% | 应较为准确 |
| 投标报价 | ±10%~25% | 关注策略性报价 |
| 结算 | ±10%~15% | 应与实际接近 |

### 3.4 检查规则

| 规则ID | 规则名称 | 检查逻辑 | 严重程度 |
|--------|----------|----------|:--------:|
| PRC-CMP-001 | 单价偏高检查 | 实际单价 > 参考价 × (1 + 上浮比例) | 🟡 一般 |

| PRC-CMP-006 | 单价为负检查 | 综合单价 < 0 | 🔴 严重 |
| PRC-CMP-007 | 无参考数据 | 未能匹配到历史综价数据 | 🔵 提示 |

## 4. 一致性检查

### 4.1 检查原理

一致性检查确保**同一造价文件内**，相同或相似清单项的综合单价保持一致或合理差异。

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          一致性检查原理                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   检查场景1：完全相同的清单项                                           │
│   ─────────────────────────────                                         │
│                                                                         │
│   某造价文件中：                                                        │
│   清单项A - 1#楼/矩形柱/C30/400×400 .......... 单价 580元/m³          │
│   清单项B - 2#楼/矩形柱/C30/400×400 .......... 单价 580元/m³  ✓       │
│   清单项C - 3#楼/矩形柱/C30/400×400 .......... 单价 520元/m³  ✗       │
│                                                                         │
│   问题：相同清单项出现不同单价，需要告警                                │
│                                                                         │
│   检查场景2：相似清单项（特征有差异）                                   │
│   ─────────────────────────────────                                     │
│                                                                         │
│   清单项A - 矩形柱/C30/400×400 .......... 单价 580元/m³               │
│   清单项B - 矩形柱/C30/500×500 .......... 单价 560元/m³               │
│   清单项C - 矩形柱/C35/400×400 .......... 单价 610元/m³               │
│                                                                         │
│   分析：特征差异导致单价差异，需判断差异是否合理                        │
│   • C30→C35：单价上涨5%，合理                                          │
│   • 400×400→500×500：单价下降3%，合理（大截面模板周转效率高）          │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 4.2 一致性判定规则

#### 4.2.1 相同清单项判定

两个清单项被视为"完全相同"需满足：

| 条件 | 必须/可选 | 说明 |
|------|:--------:|------|
| SBoQCode 相同 | 必须 | 标准清单编码一致 |
| 全部项目特征相同 | 必须 | 强度、尺寸、材料等完全一致 |
| 计量单位相同 | 必须 | - |

#### 4.2.2 相似清单项判定

两个清单项被视为"相似"需满足：

| 条件 | 说明 |
|------|------|
| SBoQCode 相同 | 标准清单编码一致 |
| 关键特征相近 | 如强度等级差≤1级，尺寸差≤20% |
| 计量单位相同 | - |

#### 4.2.3 单价差异合理性判定

| 特征差异类型 | 合理单价变化 | 判定逻辑 |
|-------------|-------------|----------|
| 混凝土强度提高1级 | +3%~8% | 材料成本增加 |
| 截面尺寸增大20% | -2%~5% | 模板效率提高/钢筋含量变化 |
| 层高增加（≤1m） | +5%~15% | 支撑体系成本增加 |
| 不同楼栋同清单 | ±2%以内 | 应基本一致 |

### 4.3 检查规则

| 规则ID | 规则名称 | 检查逻辑 | 严重程度 |
|--------|----------|----------|:--------:|
| PRC-CMP-101 | 相同清单单价不一致 | 完全相同的清单项单价不一致 | 🔴 严重 |
| PRC-CMP-102 | 相同清单单价微小偏差 | 完全相同清单项单价偏差≤2% | 🔵 提示 |
| PRC-CMP-103 | 相似清单单价差异异常 | 相似清单项单价差异超出合理范围 | 🟡 一般 |
| PRC-CMP-104 | 特征差异与单价差异不匹配 | 特征提高但单价降低（或反之） | 🟡 一般 |

## 5. 完整性检查

### 5.1 检查原理

完整性检查确保综合单价的**组成完整**、**占比合理**，防止漏计或错计。

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          完整性检查原理                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   检查维度1：组成完整性                                                 │
│   ─────────────────────                                                 │
│                                                                         │
│   综合单价应包含：                                                      │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ □ 人工费（必须）                                                 │  │
│   │ □ 材料费（必须，除纯人工项目外）                                 │  │
│   │ □ 机械费（视清单项性质，可选）                                   │  │
│   │ □ 管理费（必须）                                                 │  │
│   │ □ 利润（必须）                                                   │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   检查维度2：占比合理性                                                 │
│   ─────────────────────                                                 │
│                                                                         │
```
