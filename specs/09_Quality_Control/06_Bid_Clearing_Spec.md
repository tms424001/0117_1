## 1. 功能概述

### 1.1 功能定位

清标是质控模块的**多文件对比能力**之一，以招标控制价为基准，对多家投标文件进行符合性审查和价格分析，辅助招标人进行评标决策。

### 2.2 检查项目

#### 2.2.1 清单完整性检查

| 检查项 | 检查逻辑 | 废标风险 |
|--------|----------|:--------:|
| 清单项缺失 | 投标清单项少于控制价 | 🔴 高 |
| 清单项新增 | 投标清单项多于控制价 | 🟡 中 |
| 清单编码不一致 | 同位置清单编码不同 | 🔴 高 |
| 清单名称不一致 | 同编码清单名称不同 | 🟡 中 |

#### 2.2.2 工程量一致性检查

| 检查项 | 检查逻辑 | 废标风险 | 说明 |
|--------|----------|:--------:|------|
| 工程量修改 | 投标工程量≠控制价工程量 | 🔴 高 | 不可调整工程量 |
| 工程量偏差 | 偏差在允许范围内 | 🟡 中 | 可调整项目 |
| 工程量为零 | 投标工程量=0 | 🔴 高 | 明显异常 |

**工程量偏差判定**：

| 偏差范围 | 判定 | 说明 |
|:--------:|------|------|
| 0% | 完全一致 | 正常 |
| 0%~1% | 微小偏差 | 可能是精度差异 |
| 1%~5% | 轻度偏差 | 需确认是否允许调整 |
| >5% | 明显偏差 | 需要澄清或废标 |

#### 2.2.3 单位一致性检查

| 检查项 | 检查逻辑 | 废标风险 |
|--------|----------|:--------:|
| 单位不一致 | 投标单位≠控制价单位 | 🔴 高 |
| 单位变体 | 使用非标准单位写法 | 🔵 低 |

#### 2.2.4 其他符合性检查

| 检查项 | 检查逻辑 | 废标风险 |
|--------|----------|:--------:|
| 总价大写小写不一致 | 大写金额≠小写金额 | 🔴 高 |
| 投标总价超控制价 | 投标价>最高限价 | 🔴 高 |
| 分部汇总错误 | 子项之和≠分部合计 | 🟡 中 |
| 措施费缺项 | 必要措施费未计 | 🟡 中 |
| 规费税金异常 | 规费税金计算不符规定 | 🟡 中 |

| 规则ID | 规则名称 | 检查逻辑 | 严重程度 |
|--------|----------|----------|:--------:|
| BID-CMP-001 | 清单项缺失 | 控制价有的项，投标文件没有 | 🔴 严重 |
| BID-CMP-002 | 清单项新增 | 投标文件有的项，控制价没有 | 🟡 一般 |

| BID-CMP-007 | 计量单位不一致 | 投标单位≠控制价单位 | 🔴 严重 |
| BID-CMP-008 | 投标总价超限价 | 投标价>招标控制价 | 🔴 严重 |
| BID-CMP-009 | 大小写金额不一致 | 总价大写≠小写 | 🔴 严重 |

### 3.2 偏离率计算

#### 3.2.1 单价偏离率

```
单价偏离率 = (投标单价 - 控制价单价) / 控制价单价 × 100%
```

**偏离率判定标准**：

| 偏离范围 | 判定 | 说明 |
|:--------:|------|------|
| ±5%以内 | 正常 | 合理波动范围 |
| ±5%~15% | 轻度偏离 | 需关注 |
| ±15%~30% | 中度偏离 | 需重点关注 |
| ±30%以上 | 严重偏离 | 可能存在问题 |

#### 3.2.2 合价偏离率

```
合价偏离率 = (投标合价 - 控制价合价) / 控制价合价 × 100%
```

#### 3.2.3 加权偏离率

考虑合价权重的综合偏离：

```
加权偏离率 = Σ(单价偏离率 × 该项合价权重)
```

#### 3.3.2 不平衡报价指数

```typescript
// 不平衡报价指数计算
interface UnbalancedIndex {
  highDeviationCount: number;      // 高报价项数（偏离>30%）
  lowDeviationCount: number;       // 低报价项数（偏离<-30%）
  totalItems: number;              // 总清单项数
  
  highDeviationAmount: number;     // 高报价项合价
  lowDeviationAmount: number;      // 低报价项合价
  totalAmount: number;             // 总合价
  
  // 不平衡指数 = (高报价合价 + 低报价合价) / 总合价
  unbalancedIndex: number;
}
```

**不平衡指数判定**：

| 指数范围 | 判定 | 说明 |
|:--------:|------|------|
| <0.15 | 正常 | 报价基本平衡 |
| 0.15~0.25 | 轻度不平衡 | 有一定不平衡报价倾向 |
| 0.25~0.40 | 中度不平衡 | 明显不平衡报价 |
| >0.40 | 严重不平衡 | 存在严重不平衡报价 |

#### 3.4.2 围串标风险评分

| 特征 | 权重 | 评分标准 |
|------|:----:|----------|
| 单价一致性 | 30% | 完全一致项占比>80%：高风险 |
| 总价梯度分布 | 25% | 差距呈固定比例递增：高风险 |
| 相同错误 | 25% | 出现2处以上相同错误：高风险 |
| 投标时间相近 | 10% | 投标时间间隔<5分钟：中风险 |
| 联系方式相似 | 10% | 电话/地址相似：中风险 |

```typescript
// 清单映射算法
interface ItemMapping {
  baselineItemId: string;              // 控制价清单项ID
  baselineCode: string;                // 控制价清单编码
  baselineSeq: number;                 // 控制价序号
  
  bidMappings: {
    bidderId: string;                  // 投标人ID
    bidItemId: string | null;          // 投标清单项ID（null表示缺失）
    bidCode: string | null;            // 投标清单编码
    bidSeq: number | null;             // 投标序号
    mappingType: 'exact' | 'similar' | 'missing' | 'extra';  // 映射类型
    confidence: number;                // 映射置信度
  }[];
}

// 映射类型说明
// exact: 编码完全匹配
// similar: 编码不同但名称/特征相似
// missing: 投标文件中缺失
// extra: 投标文件中多出
```

## 8. API 接口

### 8.1 接口列表

| 接口名称 | 方法 | 路径 | 说明 |
|----------|------|------|------|
| 创建清标项目 | POST | `/api/qc/bid-clearing/projects` | 创建新的清标项目 |
| 获取项目列表 | GET | `/api/qc/bid-clearing/projects` | 获取清标项目列表 |
| 获取项目详情 | GET | `/api/qc/bid-clearing/projects/{projectId}` | 获取清标项目详情 |
| 上传基准文件 | POST | `/api/qc/bid-clearing/projects/{projectId}/baseline` | 上传招标控制价 |
| 上传投标文件 | POST | `/api/qc/bid-clearing/projects/{projectId}/bids` | 上传投标文件 |
| 创建清标轮次 | POST | `/api/qc/bid-clearing/projects/{projectId}/rounds` | 创建新的清标轮次 |
| 执行清标检查 | POST | `/api/qc/bid-clearing/projects/{projectId}/rounds/{roundId}/check` | 执行清标检查 |
| 获取检查结果 | GET | `/api/qc/bid-clearing/projects/{projectId}/rounds/{roundId}/result` | 获取清标结果 |
| 获取单价对比 | GET | `/api/qc/bid-clearing/projects/{projectId}/rounds/{roundId}/price-comparison` | 获取单价对比数据 |
| 生成清标报告 | POST | `/api/qc/bid-clearing/projects/{projectId}/rounds/{roundId}/report` | 生成清标报告 |

## 附录 B：围串标特征识别标准

| 单价高度相似 | 单价差异<1%的项占比>70% | 中 |

| 投标时间相近 | 时间间隔<5分钟 | 低 |
| 联系方式相似 | 电话号码或地址相近 | 低 |
