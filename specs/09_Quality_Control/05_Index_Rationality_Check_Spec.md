## 1. 功能概述

### 1.1 功能定位

指标合理性检查是质控模块的**宏观检查层**，通过经济指标和技术指标的分析，从整体上判断造价文件的合理性。这是对前面四类检查的补充和升华，从"见树"到"见林"。

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          指标合理性检查定位                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   检查层次                                                              │
│   ────────                                                              │
│   ┌─────────────┐                                                       │
│   │ 合规性检查  │ ◀── 格式层：编码、单位、结构                         │
│   ├─────────────┤                                                       │
│   │ 价格检查    │ ◀── 价值层：材料价格、综合单价                       │
│   ├─────────────┤                                                       │
│   │ 清单缺陷检查│ ◀── 内容层：特征、漏项、工程量                       │
│   ├─────────────┤                                                       │
│   │ 指标合理性  │ ◀── 宏观层：造价指标、含量指标（本文档）             │
│   └─────────────┘                                                       │
│                                                                         │
│   检查视角转换                                                          │
│   ────────────                                                          │
│                                                                         │
│   前四类检查：自下而上，逐项审查                                        │
│   ┌─────┬─────┬─────┬─────┬─────┐                                      │
│   │清单1│清单2│清单3│ ... │清单n│  ← 检查每个清单项                   │
│   └─────┴─────┴─────┴─────┴─────┘                                      │
│                                                                         │
│   指标检查：自上而下，整体把控                                          │
│   ┌─────────────────────────────┐                                      │
│   │         项目整体            │  ← 检查整体指标                      │
│   │  单方造价、含量、占比       │                                      │
│   └─────────────────────────────┘                                      │
│                                                                         │
│   核心价值：发现局部检查难以发现的整体性问题                            │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.2 检查策略

**分阶段实施策略**：指标检查依赖企业历史数据积累，初期以"参考提示"为主，随着数据积累逐步升级为"硬性检查"。

| 阶段 | 数据积累 | 检查策略 | 输出 |
|------|:--------:|----------|------|
| 初期 | <50个项目 | 仅供参考 | 🔵 提示级别 |
| 中期 | 50-200个项目 | 软性检查 | 🟡 一般级别 |
| 成熟期 | >200个项目 | 硬性检查 | 🔴 严重级别 |

### 1.3 指标分类

| 指标类别 | 指标名称 | 单位 | 说明 |
|----------|----------|------|------|
| **经济指标** | 单方造价 | 元/m² | 单位建筑面积造价 |
| | 单方土建造价 | 元/m² | 土建部分单方造价 |
| | 单方安装造价 | 元/m² | 安装部分单方造价 |
| | 单方装饰造价 | 元/m² | 装饰部分单方造价 |
| **含量指标** | 含钢量 | kg/m² | 单位面积钢筋用量 |
| | 含混凝土量 | m³/m² | 单位面积混凝土用量 |
| | 含砌体量 | m³/m² | 单位面积砌体用量 |
| | 含模板量 | m²/m² | 单位面积模板用量 |
| **占比指标** | 分部占比 | % | 各分部工程费占总造价比例 |
| | 人材机占比 | % | 人工费/材料费/机械费占比 |
| | 措施费占比 | % | 措施费占总造价比例 |

## 2. 经济指标检查

### 2.1 单方造价检查

#### 2.1.1 检查原理

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          单方造价检查原理                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   计算公式                                                              │
│   ────────                                                              │
│                                                                         │
│   单方造价 = 工程总造价 ÷ 建筑面积                                     │
│                                                                         │
│   示例：                                                                │
│   工程总造价：5,680万元                                                 │
│   建筑面积：28,400 m²                                                   │
│   单方造价：5,680万 ÷ 28,400 = 2,000 元/m²                             │
│                                                                         │
│   合理区间判定                                                          │
│   ────────────                                                          │
│                                                                         │
│   合理区间来源：                                                        │
│   ① 企业指标库（同类项目历史数据）                                     │
│   ② 行业参考值（造价指标手册）                                         │
│   ③ 地区发布值（造价信息）                                             │
│                                                                         │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │                                                                  │  │
│   │  1200      1600      2000      2400      2800                   │  │
│   │    │─────────│─────────│─────────│─────────│                   │  │
│   │    ↑         ↑         ↑         ↑         ↑                   │  │
│   │  严重偏低  偏低      合理      偏高     严重偏高                │  │
│   │                       ▲                                         │  │
│   │                    参考值                                       │  │
│   │                   2000元/m²                                     │  │
│   │                                                                  │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

#### 2.1.3 影响因素修正

单方造价受多种因素影响，需要在对比时进行修正：

| 影响因素 | 影响程度 | 修正方法 |
|----------|:--------:|----------|
| 地区差异 | ±15%~30% | 按地区调整系数 |
| 时间差异 | ±5%~15%/年 | 按价格指数调整 |
| 层数差异 | ±5%~15% | 高层比多层高约10% |
| 地下室 | +10%~20% | 有地下室造价更高 |
| 装修标准 | ±20%~50% | 毛坯/简装/精装差异大 |
| 设备配置 | ±10%~30% | 智能化程度影响 |

## 3. 含量指标检查

### 3.1 含钢量检查

#### 3.1.1 检查原理

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          含钢量检查原理                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   计算公式                                                              │
│   ────────                                                              │
│                                                                         │
│   含钢量 = 钢筋总用量(kg) ÷ 建筑面积(m²)                               │
│                                                                         │
│   示例：                                                                │
│   钢筋总用量：1,420 t = 1,420,000 kg                                    │
│   建筑面积：28,400 m²                                                   │
│   含钢量：1,420,000 ÷ 28,400 = 50 kg/m²                                │
│                                                                         │
│   含钢量的意义                                                          │
│   ──────────────                                                        │
│                                                                         │
│   • 反映结构设计的经济性                                                │
│   • 含钢量过高：可能设计偏保守或计算错误                               │
│   • 含钢量过低：可能漏算钢筋或结构不安全                               │
│                                                                         │
│   影响因素                                                              │
│   ────────                                                              │
│   • 结构形式：剪力墙 > 框架剪力墙 > 框架                               │
│   • 建筑高度：高层 > 多层 > 低层                                       │
│   • 抗震等级：高烈度区 > 低烈度区                                      │
│   • 荷载要求：工业厂房 > 普通民用                                      │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 4. 占比指标检查

```
┌─────────────────────────────────────────────────────────────────────────┐

│                                                                         │

│   ──────────────────────────────                                        │
│                                                                         │
│   工程总造价构成示例（住宅）：                                          │
│   ┌─────────────────────────────────────────────────────────────────┐  │

│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
```

#### 4.2.1 人材机占比参考值

#### 4.2.2 占比异常分析

| 异常情况 | 可能原因 | 检查建议 |
|----------|----------|----------|
| 人工费占比过高 | 定额人工单价偏高；机械化程度低 | 检查人工单价和定额选择 |
| 人工费占比过低 | 人工费漏计；机械化程度高 | 检查是否遗漏人工 |
| 材料费占比过高 | 材料价格偏高；主材占比大 | 检查材料价格合理性 |
| 材料费占比过低 | 材料漏计；价格偏低 | 检查主材是否齐全 |
| 机械费占比过高 | 大型机械使用多；台班单价高 | 检查机械配置合理性 |
| 机械费占比过低 | 机械费漏计 | 检查土方、桩基等项目 |

## 6. 规则配置

### 6.1 规则数据结构

```typescript
interface IndexCheckRule {
  // 基本信息
  ruleId: string;
  ruleName: string;
  ruleType: 'economic' | 'content' | 'ratio';  // 经济/含量/占比
  
  // 规则定义
  description: string;
  indexName: string;            // 指标名称
  indexFormula: string;         // 计算公式
  
  // 参考值配置
  referenceConfig: {
    source: 'enterprise' | 'industry' | 'custom';  // 参考值来源
    matchFields: string[];      // 匹配字段（建筑类型、结构形式等）
    defaultRange: { min: number; max: number };  // 默认参考范围
  };
  
  // 偏离阈值
  thresholds: {
    warningUpper: number;       // 上限告警系数
    warningLower: number;       // 下限告警系数
    errorUpper: number;         // 上限错误系数
    errorLower: number;         // 下限错误系数
  };
  
  // 严重程度
  severity: 'error' | 'warning' | 'info';
  
  // 适用范围
  applicableBuildingTypes: string[];
  applicableFileTypes: FileType[];
  
  // 状态
  isEnabled: boolean;
  maturityLevel: 'reference' | 'soft' | 'hard';  // 成熟度级别
  version: string;
}

// 指标参考值配置
interface IndexReference {
  buildingType: string;         // 建筑类型
  structureType: string;        // 结构形式
  floorRange: { min: number; max: number };  // 层数范围
  
  // 经济指标参考
  unitCost: { min: number; max: number; unit: string };
  
  // 含量指标参考
  steelContent: { min: number; max: number; unit: string };
  concreteContent: { min: number; max: number; unit: string };
  
  // 占比指标参考
  divisionRatios: {
    civil: { min: number; max: number };
    decoration: { min: number; max: number };
    installation: { min: number; max: number };
    measures: { min: number; max: number };
  };
  
  // 数据来源
  source: string;
  sampleCount: number;
  updateTime: Date;
}
```

## 附录 B：指标异常诊断指南

| 异常表现 | 可能原因 | 检查建议 |
|----------|----------|----------|
| 单方造价偏高 | 材料价格高；设计标准高；工程量多算 | 检查主材价格、设计标准 |
| 单方造价偏低 | 漏项；价格偏低；面积计算错误 | 检查清单完整性、价格 |
| 含钢量偏高 | 设计保守；抗震等级高；计算错误 | 核实设计图纸、工程量 |
| 含钢量偏低 | 钢筋漏算；计量错误 | 检查钢筋工程量 |
| 土建占比过高 | 装饰安装漏项；土建价格偏高 | 检查其他分部完整性 |
| 安装占比过高 | 设备配置高；价格偏高 | 检查设备清单和价格 |
| 措施费占比过高 | 费率取值偏高；特殊措施多 | 检查措施费取费标准 |
| 材料费占比过高 | 材价高；人机费漏计 | 检查材料价格和人机费 |
