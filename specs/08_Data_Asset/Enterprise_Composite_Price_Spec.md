## 1. 功能概述

### 1.1 功能定位

企业综价库是企业数据资产的核心组成部分，汇聚经过标准化处理和审核的清单综合单价数据，为计价、质控等应用模块提供高质量的清单单价参考和组价支撑。

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          企业综价库定位                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   ┌─────────────┐        ┌─────────────┐        ┌─────────────┐        │
│   │   我的综价   │  ──▶  │   PR审核    │  ──▶  │  企业综价库  │        │
│   │  (个人数据)  │       │  (质量把关)  │       │  (企业资产)  │        │
│   └─────────────┘        └─────────────┘        └──────┬──────┘        │
│                                                        │               │
│                          ┌─────────────────────────────┼───────┐       │
│                          │                             │       │       │
│                          ▼                             ▼       ▼       │
│                   ┌───────────┐              ┌──────────┐ ┌──────────┐ │
│                   │  计价模块  │              │ 质控模块 │ │ 估算模块 │ │
│                   │ (智能组价) │              │(单价对标)│ │(清单估算)│ │
│                   └───────────┘              └──────────┘ └──────────┘ │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.2 综价概念说明

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          综合单价构成                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                      清单项目综合单价                            │   │
│  │  ════════════════════════════════════════════════════════════  │   │
│  │                                                                 │   │
│  │    ┌─────────────┐                                             │   │
│  │    │   人工费    │  ← 定额人工 × 人工单价                       │   │
│  │    ├─────────────┤                                             │   │
│  │    │   材料费    │  ← Σ(材料用量 × 材料单价)                    │   │
│  │    ├─────────────┤                                             │   │
│  │    │   机械费    │  ← Σ(机械台班 × 机械单价)                    │   │
│  │    ├─────────────┤                                             │   │
│  │    │   管理费    │  ← (人工费+机械费) × 管理费率                │   │
│  │    ├─────────────┤                                             │   │
│  │    │   利润      │  ← (人工费+机械费) × 利润率                  │   │
│  │    ├─────────────┤                                             │   │
│  │    │   风险费    │  ← 分部分项合计 × 风险费率（可选）            │   │
│  │    └─────────────┘                                             │   │
│  │           ‖                                                     │   │
│  │           ▼                                                     │   │
│  │    ┌─────────────┐                                             │   │
│  │    │  综合单价   │  = 人工费+材料费+机械费+管理费+利润+风险费    │   │
│  │    └─────────────┘                                             │   │
│  │                                                                 │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  数据来源：造价文件解析 → 清单项目 → 综合单价提取                        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.3 核心价值

| 维度 | 说明 |
|------|------|
| **单价沉淀** | 将分散的清单综合单价汇聚为企业级价格资产 |
| **标准化管理** | 基于SBOQCode标准清单编码，实现清单归一化 |
| **智能组价** | 为新项目计价提供历史组价参考和快速套用 |
| **单价分析** | 单价趋势分析、构成分析、对比分析 |
| **质控支撑** | 为造价审核提供单价对标基准，识别异常报价 |

## 5. 业务规则

### 5.1 数据入库规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| ECP-001 | PR审核前置 | 综价必须通过PR审核才能入库 |
| ECP-002 | SBOQCode必填 | 必须映射到标准清单编码 |
| ECP-003 | 综合单价校验 | 综合单价 = 人工费+材料费+机械费+管理费+利润（允许±0.01误差） |
| ECP-004 | 时空维度必填 | 材价期、省份、城市、定额版本为必填项 |
| ECP-005 | 组价完整性 | 有定额明细的视为完整组价，标记is_complete=true |
| ECP-006 | 去重校验 | 同一来源的相同综价不可重复入库 |

### 5.2 单价异常检测规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| ECP-101 | 偏离度检测 | 偏离同类均值±30%标记为warning |
| ECP-102 | 极值检测 | 偏离同类均值±50%标记为error |
| ECP-103 | 构成异常 | 人材机占比与同类差异超过±15%预警 |
| ECP-104 | 负值检测 | 任一费用项为负值标记为error |
| ECP-105 | 异常排除 | 异常综价默认不参与聚合计算 |

### 5.3 综价聚合规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| ECP-201 | 聚合维度 | 按SBOQCode+特征组合+地区+材价期四维聚合 |
| ECP-202 | 特征匹配 | 相同特征组合的综价才聚合在一起 |
| ECP-203 | 最小样本数 | 样本数≥3才生成聚合结果 |
| ECP-204 | 异常排除 | quality_flag=error的数据不参与聚合 |
| ECP-205 | 推荐价选取 | 默认取中位价作为推荐价 |
| ECP-206 | 定时计算 | 每日凌晨自动增量计算 |

### 5.4 组价模板规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| ECP-301 | 模板类型 | personal个人可见，department部门可见，enterprise全企业可见 |
| ECP-302 | 模板套用 | 套用时可调整材价期、地区、定额版本等参数 |
| ECP-303 | 使用统计 | 每次套用增加use_count计数 |
| ECP-304 | 推荐逻辑 | 使用次数≥50且近30天有使用的模板标记为推荐 |
| ECP-305 | 模板编辑 | 个人模板可编辑删除，部门/企业模板仅管理员可编辑 |

### 5.5 查询筛选规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| ECP-401 | 分类筛选 | 支持清单大类→中类→小类三级联动 |
| ECP-402 | 特征筛选 | 支持按项目特征（如土质、深度）筛选 |
| ECP-403 | 地区筛选 | 支持省→市联动筛选 |
| ECP-404 | 时间范围 | 材价期支持范围筛选，默认近6个月 |
| ECP-405 | 模糊搜索 | 支持清单名称、项目特征模糊搜索 |
