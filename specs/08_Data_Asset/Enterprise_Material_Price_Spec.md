## 1. 功能概述

### 1.1 功能定位

企业材价库是企业数据资产的核心组成部分，汇聚经过标准化处理和审核的材料价格数据，为计价、质控等应用模块提供高质量的材料价格参考和对标基准。

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          企业材价库定位                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   ┌─────────────┐        ┌─────────────┐        ┌─────────────┐        │
│   │   我的材价   │  ──▶  │   PR审核    │  ──▶  │  企业材价库  │        │
│   │  (个人数据)  │       │  (质量把关)  │       │  (企业资产)  │        │
│   └─────────────┘        └─────────────┘        └──────┬──────┘        │
│                                                        │               │
│                          ┌─────────────────────────────┼───────┐       │
│                          │                             │       │       │
│                          ▼                             ▼       ▼       │
│                   ┌───────────┐              ┌──────────┐ ┌──────────┐ │
│                   │  计价模块  │              │ 质控模块 │ │ 估算模块 │ │

│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.3 核心价值

| 维度 | 说明 |
|------|------|
| **价格沉淀** | 将分散的材料价格汇聚为企业级价格资产 |
| **标准化管理** | 基于SMLCode标准材料编码，实现材料归一化 |
| **多维查询** | 按材料分类、地区、时间、供应商等多维度查询 |
| **价格分析** | 价格趋势分析、波动预警、供应商比价 |
| **计价支撑** | 为智能套定额、组价提供材料价格支撑 |

|----------|----------|----------|
| EMP-001 | PR审核前置 | 材价必须通过PR审核才能入库 |
| EMP-002 | SMLCode必填 | 必须映射到标准材料编码 |
| EMP-003 | 时空维度必填 | 材价期、省份、城市为必填项 |
| EMP-004 | 价格有效性 | 单价必须大于0 |
| EMP-005 | 来源信息 | 必须填写数据来源和来源类型 |
| EMP-006 | 去重校验 | 同一来源的相同材价不可重复入库 |

### 5.2 价格异常检测规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| EMP-101 | 偏离度检测 | 偏离同类均值±30%标记为warning |
| EMP-102 | 极值检测 | 偏离同类均值±50%标记为error |
| EMP-103 | 波动检测 | 月度环比波动超过±20%触发预警 |
| EMP-104 | 历史比对 | 与历史同期价格偏差超过±30%预警 |
| EMP-105 | 异常排除 | 异常价格默认不参与聚合计算 |

### 5.3 价格聚合规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| EMP-201 | 聚合维度 | 按SMLCode+地区+材价期三维聚合 |
| EMP-202 | 最小样本数 | 样本数≥3才生成聚合结果 |
| EMP-203 | 异常排除 | quality_flag=error的数据不参与聚合 |
| EMP-204 | 推荐价选取 | 默认取中位价作为推荐价 |
| EMP-205 | 价格区间 | 默认取25分位~75分位作为价格区间 |
| EMP-206 | 定时计算 | 每日凌晨自动增量计算 |

### 5.4 基准价管理规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| EMP-301 | 版本号规则 | 格式：YYYY-MM-VN（如2025-06-V1） |
| EMP-302 | 审核前置 | 基准价发布必须经过审核 |
| EMP-303 | 有效期管理 | 基准价默认有效期3个月 |
| EMP-304 | 覆盖规则 | 新版本发布后旧版本自动失效 |
| EMP-305 | 回溯支持 | 支持查询历史版本基准价 |

### 5.5 查询筛选规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| EMP-401 | 分类筛选 | 支持材料大类→中类→小类三级联动 |
| EMP-402 | 地区筛选 | 支持省→市→区三级联动 |
| EMP-403 | 时间范围 | 材价期支持范围筛选，默认近6个月 |
| EMP-404 | 模糊搜索 | 支持材料名称、规格型号模糊搜索 |
| EMP-405 | 组合筛选 | 多条件AND组合 |

## 7. 状态机

### 7.1 材价状态流转

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          材价状态机                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│                    PR审核通过                                           │
│                        │                                                │
│                        ▼                                                │
│                  ┌──────────┐                                           │
│                  │  active  │ ◀──────────────────────┐                  │
│                  │  (启用)   │                        │                  │
│                  └────┬─────┘                        │                  │
│                       │                              │                  │
│          ┌────────────┼────────────┐                │                  │
│          │            │            │                │                  │
│          │ 手动停用    │ 超过有效期  │ 重新启用       │                  │
│          ▼            ▼            │                │                  │
│    ┌──────────┐ ┌──────────┐      │                │                  │
│    │ inactive │ │ expired  │──────┘                │                  │
│    │  (停用)   │ │  (过期)   │                       │                  │
│    └────┬─────┘ └──────────┘                       │                  │
│         │                                          │                  │
│         │ 确认删除                                  │                  │
│         ▼                                          │                  │
│    ┌──────────┐                                    │                  │
│    │ deleted  │────────────────────────────────────┘                  │
│    │ (已删除)  │  (特殊情况可恢复)                                      │
│    └──────────┘                                                        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 7.2 状态说明

| 状态 | 编码 | 说明 | 可执行操作 |
|------|------|------|------------|
| 启用 | active | 正常状态，可查询、可引用 | 停用、核实、导出 |
| 停用 | inactive | 暂停使用，不出现在推荐结果 | 启用、删除 |
| 过期 | expired | 超过有效期，可查询但标记过期 | 重新启用、删除 |
| 已删除 | deleted | 软删除，数据保留但不可见 | 恢复（管理员） |
