## 1. 功能概述

### 1.1 功能定位

公共资源案例库是数据商城的核心数据资产之一，汇聚平台脱敏公开的案例数据、政府公开的招投标案例以及行业公开数据，为用户提供跨企业、跨地区的行业参考案例，支撑估算、对标等业务场景。

### 1.3 核心价值

| 维度 | 说明 |
|------|------|
| **行业参考** | 提供跨企业的行业案例参考，扩大用户视野 |
| **估算支撑** | 为投资估算提供行业指标基准和参考案例 |
| **跨企对标** | 支持与行业数据对标，识别自身差距 |
| **市场洞察** | 反映区域市场造价水平和变化趋势 |
| **知识共享** | 构建行业知识共享平台，促进行业发展 |

## 5. 业务规则

### 5.1 数据入库规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| PCS-001 | 来源审核 | 所有案例必须经过审核才能发布 |
| PCS-002 | 脱敏要求 | 平台贡献案例必须完成敏感信息脱敏 |
| PCS-003 | 完整度要求 | 数据完整度≥70%才能入库 |
| PCS-004 | 标签必填 | 功能标签、规模档、地区为必填项 |
| PCS-005 | 去重校验 | 相同来源的相同案例不可重复入库 |
| PCS-006 | 质量分级 | 入库时自动计算质量等级（A/B/C/D） |

### 5.2 脱敏规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| PCS-101 | 项目名称脱敏 | 移除具体项目名称，保留功能描述（如"某市级办公楼"） |
| PCS-102 | 单位名称脱敏 | 建设单位、施工单位、设计单位等全部脱敏 |
| PCS-103 | 地址脱敏 | 精确到区县级别，不显示详细地址 |
| PCS-104 | 金额保留 | 造价金额保留，但不显示合同编号等 |
| PCS-105 | 联系方式脱敏 | 所有联系方式、联系人信息全部移除 |
| PCS-106 | 文件脱敏 | 清单明细、定额明细等详细数据根据授权范围决定 |

### 5.3 质量分级规则

| 等级 | 编码 | 完整度要求 | 数据要求 | 可信度要求 |
|------|------|------------|----------|------------|
| A级 | A | ≥90% | 含详细指标数据 | 平台审核 |
| B级 | B | ≥80% | 含主要指标数据 | 来源可查 |
| C级 | C | ≥70% | 含基本造价数据 | 数据自洽 |
| D级 | D | <70% | 仅供参考 | 无法验证 |

### 5.4 访问权限规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| PCS-301 | 公开访问 | access_level=public的案例所有用户可见 |
| PCS-302 | 会员访问 | access_level=member的案例需普通会员权限 |
| PCS-303 | VIP访问 | access_level=vip的案例需VIP会员权限 |
| PCS-304 | 导出权限 | 详细数据导出需会员权限 |
| PCS-305 | 浏览限制 | 非会员每日浏览上限50个案例 |

### 5.5 贡献积分规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| PCS-401 | A级积分 | 贡献A级案例获得500积分 |
| PCS-402 | B级积分 | 贡献B级案例获得300积分 |
| PCS-403 | C级积分 | 贡献C级案例获得150积分 |
| PCS-404 | 审核通过 | 积分在审核通过后发放 |
| PCS-405 | 积分有效期 | 积分永久有效，不清零 |
| PCS-406 | 积分兑换 | 积分可兑换会员时长、服务等 |

### 5.6 行业基准计算规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| PCS-501 | 聚合维度 | 按功能标签+规模档+地区+年份聚合 |
| PCS-502 | 最小样本 | 样本数≥5才生成基准指标 |
| PCS-503 | 异常排除 | 质量等级D的案例不参与基准计算 |
| PCS-504 | 推荐值 | 默认取中位数作为推荐值 |
| PCS-505 | 定期更新 | 每月1日自动重新计算基准指标 |
