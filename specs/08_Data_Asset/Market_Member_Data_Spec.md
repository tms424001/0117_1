## 1. 功能概述

### 1.1 功能定位

企业案例库是企业数据资产的核心组成部分，存储经过标准化处理和审核的项目案例数据，为质控、计价、估算等应用模块提供高质量的参考数据支撑。

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          企业案例库定位                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   ┌─────────────┐        ┌─────────────┐        ┌─────────────┐        │
│   │   我的项目   │  ──▶  │   PR审核    │  ──▶  │  企业案例库  │        │
│   │  (个人数据)  │       │  (质量把关)  │       │  (企业资产)  │        │
│   └─────────────┘        └─────────────┘        └──────┬──────┘        │
│                                                        │               │
│                          ┌─────────────────────────────┼───────┐       │
│                          │                             │       │       │
│                          ▼                             ▼       ▼       │
│                   ┌───────────┐              ┌──────────┐ ┌──────────┐ │
│                   │  估算模块  │              │ 质控模块 │ │ 计价模块 │ │
│                   │ (指标参考) │              │(对标分析)│ │(价格参考)│ │
│                   └───────────┘              └──────────┘ └──────────┘ │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.2 核心价值

| 维度 | 说明 |
|------|------|
| **数据资产化** | 将分散的项目数据转化为可复用的企业级数据资产 |
| **知识沉淀** | 积累企业造价经验，形成可查询、可分析的知识库 |
| **决策支撑** | 为投资估算、成本控制提供历史数据参考 |
| **质量保障** | 经过PR审核，确保数据质量和一致性 |

| EC-003 | 标签必填 | 功能标签、规模档为必填项，无法自动匹配时必须手工选择 |
| EC-004 | 造价数据校验 | 总造价 = 分部分项 + 措施项目 + 其他项目 + 规费 + 税金，允许±0.01%误差 |
| EC-005 | 单方造价校验 | 单方造价 = 总造价 / 建筑面积，自动计算并校验 |
| EC-006 | 去重校验 | 同一来源项目不可重复入库，以source_project_id判断 |
| EC-007 | 版本控制 | 同一项目多次入库时，版本号自增，保留历史版本 |

### 5.2 数据质量规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| EC-101 | 完整度计算 | 完整度 = 已填字段数 / 总字段数 × 100% |
| EC-102 | 质量评分 | 综合评分 = 完整度×40% + 规范性×30% + 准确性×30% |
| EC-103 | 质量等级划分 | A级:≥85分，B级:70-84分，C级:60-69分，D级:<60分 |
| EC-104 | 异常值标记 | 指标值偏离同类均值±50%时，标记为warning |
| EC-105 | 必填字段校验 | 必填字段为空时，数据完整度扣减对应权重 |

### 5.3 查询筛选规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| EC-201 | 功能标签筛选 | 支持大类→中类→标签三级联动筛选 |
| EC-202 | 规模档筛选 | 支持规模类型+规模档联动筛选 |
| EC-203 | 地区筛选 | 支持省→市→区三级联动筛选 |
| EC-204 | 时间范围 | 入库时间筛选，默认近1年 |
| EC-205 | 组合筛选 | 多条件AND组合，同一条件内多选为OR |
| EC-206 | 模糊搜索 | 支持案例名称、项目名称、建设单位模糊搜索 |

### 5.4 对比分析规则

| 规则编号 | 规则名称 | 规则说明 |
|----------|----------|----------|
| EC-301 | 对比数量限制 | 单次对比最多5个案例 |
| EC-302 | 对比条件建议 | 建议选择同功能标签、同规模档案例进行对比 |
| EC-303 | 均值计算 | 对比时自动计算所选案例的各项指标均值 |
| EC-304 | 偏差计算 | 偏差 = (案例值 - 均值) / 均值 × 100% |

## 7. 状态机

### 7.1 案例状态流转

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          案例状态机                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│                    PR审核通过                                           │
│                        │                                                │
│                        ▼                                                │
│                  ┌──────────┐                                           │
│                  │  active  │ ◀──────────────────────┐                  │
│                  │  (启用)   │                        │                  │
│                  └────┬─────┘                        │                  │
│                       │                              │                  │
│          ┌────────────┼────────────┐                │                  │
│          │            │            │                │                  │
│          │ 手动停用    │ 超过有效期  │ 重新启用       │                  │
│          ▼            ▼            │                │                  │
│    ┌──────────┐ ┌──────────┐      │                │                  │
│    │ inactive │ │ expired  │──────┘                │                  │
│    │  (停用)   │ │  (过期)   │ (管理员可重新启用)     │                  │
│    └────┬─────┘ └──────────┘                       │                  │
│         │                                          │                  │
│         │ 确认删除                                  │                  │
│         ▼                                          │                  │
│    ┌──────────┐                                    │                  │
│    │ deleted  │                                    │                  │
│    │ (已删除)  │ ──────────────────────────────────┘                  │
│    └──────────┘   (特殊情况可恢复，需管理员操作)                        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 7.2 状态说明

| 状态 | 编码 | 说明 | 可执行操作 |
|------|------|------|------------|
| 启用 | active | 正常状态，可查询、可引用、参与统计 | 停用、导出、引用、修订 |
| 停用 | inactive | 暂停使用，不出现在查询结果 | 启用、删除 |
| 过期 | expired | 超过有效期，可查询但不参与统计 | 重新启用、删除 |
| 已删除 | deleted | 软删除，数据保留但不可见 | 恢复（管理员） |
